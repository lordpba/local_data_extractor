📄 Local Data Extractor — Help

Quick Start

  • Select a vision model (gemma3:27b recommended for best quality)
  • Upload one or more documents (PDF, JPG, PNG)
  • Choose a preset template or define custom fields
  • Click Extract Data and review results in the validation panel
  • Edit any field, click Validate, then Export to Excel

Extraction Strategies

  • Auto — Picks the best strategy based on the model and settings
  • Single Pass — Sends image directly to the vision model for JSON extraction (fastest)
  • OCR → Extract — Phase 1: vision OCR reads all text. Phase 2: text-only LLM extracts structured data (best quality for handwriting)

Handwriting Mode

Enable for handwritten documents. Adds extra prompts for character-by-character attention and reduces hallucination on ambiguous characters.

Template Presets

Use built-in presets for common Italian documents: Fattura, CV, Carta d'Identità, Contratto. You can also save your custom field configurations to the browser's local storage.

Tips

  • For multi-page PDFs, use Page Range to process only relevant pages
  • Higher-parameter models (27B) are slower but significantly more accurate
  • The OCR → Extract strategy works best with models that struggle with direct JSON extraction
  • gemma3:27b scored 81% on handwritten CF tests — the best local result
